# Online Voting System 🚀

This is a mini project developed using C programming language for managing and conducting online voting digitally.

---

## ✨ Features
- Admin Login
- Voter Authentication
- Vote Casting
- Vote Counting
- Result Display

---

## 🛠️ Technologies Used
- C Programming
- File Handling
- Structures

---

## 📌 Project Objective
The main objective of this project is to create a simple and secure voting system using C programming concepts.
